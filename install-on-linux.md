# Install JAP on Linux

## 1. install zsh
```sh
sudo apt update
sudo apt install zsh
```

## 2. install jq
```sh
sudo apt install jq
```

## 3. install JAP

install jap with the install command [JAP Install](https://github.com/philipstuessel/jap)

## 4. add this in .bashrc
```sh
source $HOME/.zshrc
```

## 5. update .bashrc
```sh
source .bashrc 
```
